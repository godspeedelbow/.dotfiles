.atom
=============

Install
-------------

- Create Atom folder: `mkdir ~/.atom`
- Create symlink: `ln -s ~/.dotfiles/.atom/* ~/.atom/``

Update packages
-------------

- To make a snapshot, run `make packages.list`.
- To update packages, run `make install-all-packages`.
